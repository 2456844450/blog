# blog
be based on vue.js and flask build personal blog system
> 个人博客

### 项目已部署，在线Demo
- 前台博客：http://jiushianzhoua.top/
- 后台管理系统: http://jiushianzhoua.top/permission/login (账号:anzhou,密码:123456)

### 前端所用技术
- 后台页面
    - axios
    - element-ui
    - highlight.js
    - marked
    - nprogress
    - qs
    - simplemde
    - vue
    - vue-router
    - vuex
- 前台页面
    - axios
    - element-ui
    - highlight.js
    - marked
    - vue
    - vue-class-component
    - vue-property-decorator
    - vue-router
    - vuex
 ### 后端所用技术
 - flask
 - flask-cors
 - flask-sqlalchemy
 - itsdangerous
 - werkzeug
 - wtforms
